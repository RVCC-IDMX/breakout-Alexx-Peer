Breakout Game Project

Welcome to the documentation for the Breakout Game project. This tutorial series will guide you through creating a classic Breakout game using ES6 classes and HTML5 Canvas.

Assignment Overview

In this assignment, you'll implement a classic Breakout game, building on provided starter code. You'll create the core game mechanics including rendering game elements on canvas, handling paddle movement, implementing ball physics, and managing collisions.

The assignment includes:

  • Implementing ES6 classes for game entities
  • Creating Canvas-based rendering
  • Building collision detection systems
  • Managing game state and user interaction
  • Working with a modular JavaScript architecture
